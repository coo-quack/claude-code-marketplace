# coo-quack claude-code-marketplace

Claude Code plugin marketplace for coo-quack.

## Usage

```bash
claude plugin marketplace add coo-quack/claude-code-marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [sensitive-canary](https://github.com/coo-quack/sensitive-canary) | Blocks secrets and PII before they reach the Anthropic API |
| [do-not-compact](https://github.com/coo-quack/do-not-compact) | Reloads CLAUDE.md files into context after conversation compaction |

## Install a plugin

```bash
claude plugin install sensitive-canary@coo-quack/claude-code-marketplace
claude plugin install do-not-compact@coo-quack/claude-code-marketplace
```
